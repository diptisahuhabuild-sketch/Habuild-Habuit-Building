Google Sheet Setup (5 minutes)

  1. Create a new Google Sheet
  2. Add headers in row 1: Name | Phone | Score | Date | Answers | ID
  3. Go to Extensions → Apps Script
  4. Replace the code with: 
    function doPost(e) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  var data = JSON.parse(e.postData.contents);
  sheet.appendRow([
    data.name || '',
    (data.cc||'') + (data.phone||''),
    data.score != null ? data.score : '',
    data.date || '',
    data.answers ? data.answers.join(';') : '',
    data.id || ''
  ]);
  return ContentService.createTextOutput('OK');
}
  5. Click Deploy → New deployment → Web app
  6. Set "Execute as" = Me, "Who has access" = Anyone
  7. Copy the URL and paste it above
  8. Share the Google Sheet with your email for edit access
